Все основные команды netrw (файлового менеджера)
% - создать файл
d - создать папку (директорию)
D - удалить файл/папку
R - переименовать
Enter - Открыть файл/папку
- - Подняться на уровень выше
gh - Скрыть/показать скрытые файлы (.gitignore и т.д.)
I - Скрыть/показать баннер (верхняя инструкция)
s Сортировать (нажимайте несколько раз — по имени, времени, размеру)
o - Открыть файл в горизонтальном сплите
v - Открыть файл в вертикальном сплите
ctrl-w + h/l - Перемещаться между окнами (h — налево, l — направо)

:Vexplore - открыть вертикальный сплит с netrw
:Sexplore - открыть горизонтальный сплит с netrw
