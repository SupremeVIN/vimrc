# Основные команды netrw (файлового менеджера Vim)

| Команда | Действие |
|---------|----------|
| `%` | Создать файл |
| `d` | Создать папку (директорию) |
| `D` | Удалить файл/папку |
| `R` | Переименовать |
| `Enter` | Открыть файл/папку |
| `-` | Подняться на уровень выше |
| `gh` | Скрыть/показать скрытые файлы (`.gitignore` и т.д.) |
| `I` | Скрыть/показать баннер (верхняя инструкция) |
| `s` | Сортировать (нажимайте несколько раз — по имени, времени, размеру) |
| `o` | Открыть файл в горизонтальном сплите |
| `v` | Открыть файл в вертикальном сплите |
| `Ctrl-w` + `h` / `l` | Перемещаться между окнами (`h` — налево, `l` — направо) |

## Команды для открытия netrw

| Команда | Действие |
|---------|----------|
| `:Vexplore` | Открыть вертикальный сплит с netrw |
| `:Sexplore` | Открыть горизонтальный сплит с netrw |

## Работа с файлами и папками через терминал

Некоторые действия удобнее выполнять в отдельной вкладке терминала (не выходя из Vim):

**Откройте вторую вкладку терминала** и используйте стандартные команды Linux:

| Команда | Действие |
|---------|----------|
| `touch новый_файл.py` | Создать файл |
| `mkdir имя_папки` | Создать папку |
| `mv файл.py папка/` | Переместить файл в папку |
| `cp файл.py файл_копия.py` | Скопировать файл |
| `rm файл.py` | Удалить файл |
| `rm -r папка/` | Удалить папку с содержимым |

### Как организовать работу

1. Откройте терминал
2. Запустите Vim: `vim .`
3. Откройте **вторую вкладку** терминала (Ctrl+Shift+T в большинстве терминалов)
4. В первой вкладке — Vim с файловым менеджером
5. Во второй вкладке — обычная командная строка для операций с файлами
6. Переключайтесь между вкладками (обычно Alt+1, Alt+2 или Ctrl+PageUp/PageDown)
